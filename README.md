# DougieWougie WordPress Theme

This is a modern WordPress theme for personal blogs.

## Quick Start
1. Move this folder to `wp-content/themes/`.
2. Activate from the WordPress dashboard.

## Features
- **Gutenberg Ready:** Full support for the WordPress block editor with custom styles and patterns.
- **Dark Mode:** Integrated dark mode with a toggle switch.
- **Responsive Layout:** Optimized for all screen sizes.
- **Modern Tech:** Uses CSS Custom Properties and Modern JavaScript (IE11 support is dropped).
- **Animations:** Subtle entrance animations and interactive elements.
- **Custom Patterns:** Includes a "Contact Card" block pattern.

## Browser Support
This theme is designed for modern browsers (Chrome, Firefox, Safari, Edge). **IE11 is not supported** due to the use of CSS Variables and Modern JavaScript APIs.
